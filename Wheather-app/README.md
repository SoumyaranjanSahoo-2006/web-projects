
# ☁️ Weather Explorer

A **fully responsive weather application** that displays real-time weather conditions and a 4-day forecast for any city using the **OpenWeather API**.

---

## 🌟 Features
- 🔍 Search weather by city name
- 🌡️ Displays temperature and description
- 📅 4-day weather forecast
- 🖼️ Dynamic background based on weather
- 📱 Fully mobile responsive
- ⚡ Fast and clean UI

---

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Async/Await)
- OpenWeather API

## 🔑 API Setup (Important)
This project uses the **OpenWeather API**.

### Steps:
1. Go to https://openweathermap.org
2. Create an account
3. Generate your API key
4. Replace this line in `script.js`:
   ```js
   const apiKey = "YOUR_API_KEY";
